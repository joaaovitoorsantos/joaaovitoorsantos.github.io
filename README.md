# Joăo Vitor dos Santos — Portfólio

Portfólio pessoal de Joăo Vitor dos Santos, Desenvolvedor Full Stack com foco em Next.js, TypeScript e React Native.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router DOM

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

O site é publicado automaticamente no GitHub Pages a cada push na branch `main` via GitHub Actions.

URL: https://joaaovitoorsantos.github.io
